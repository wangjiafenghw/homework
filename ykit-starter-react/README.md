# ykit-starter-react
YKit-React 示例工程

## Getting Started
```
$ git clone https://github.com/roscoe054/ykit-starter-react.git
$ cd ykit-starter-react
$ npm install 或者 yarn
```

## Run Dev
```
$ npm start
```
访问 http://127.0.0.1:3000/index.html

## Build
```
$ ykit pack -m
```

访问 https://ykit.ymfe.org 来获取更多文档
